event : 事件对象

当事件发生的时候，存储着和事件相关的一些信息的对象
    关于event的兼容性：
             e8- / chrome / ie9+  :  window.event
             火狐 / chrome / ie9+ ： 事件函数的第一个形参
    比较重要属性：
            clientX/clientY : 鼠标距离浏览器窗口的距离
            pageX/pageY : 鼠标距离页面的距离
    兼容写法：
          document.onclick = function(ev){
                ev = ev || window.event;
                        ｝
添加事件：
         obj.onclick = function(){};
  这是最常见的一种 但是同一个对象的同样事件无法执行多次
                         例：
                        obj.onclick = function(){alert(1)};
                        obj.onclick = function(){alert(2)};
                        这里只会弹框2 因为这样相当于等号赋值 所以会覆盖掉第一个
                        要多次添加事件只能用attachEvent()/addEventListener


        attachEvent()（ie）          例：oBox.attachEvent( 'onclick' , function(){
                             alert( this );
                         });   
                         ！！注意 attachEvent（） this指向window 但是我们希望是指向oBox
                         所以应该这样写：
                                 obj.attachEvent( 'on' + eventName , function(){          
                            eventFn.call(obj);
                         });       eventFn这里是一个随便定义的方法


        addEventlistener() (chrome)  例：obj.addEventListener( eventName , eventFn , false/true );
                          addEventlistener()的this 指向obj 没有问题 false/true与捕获有关

        兼容写法：
        bind( oBox , 'click' , function(){
                alert( 1 );
            });
        function bind( obj , eventName , eventFn ){            
             obj.attachEvent?obj.attachEvent('on'+eventName,function(){eventFn.call(obj);}):obj.addEventListener(eventName,eventFn,false);
        };  注意 三目运算不能换行！

DOM节点相关操作:

NodeType:节点类型有12种（常见的有 元素节点1、文本节点3、属性节点2）；

             可以用 .length得到长度；
children：（只包含元素节点，推荐使用）；

lastChild / lastElementChild : （最后一个子节点）；
nextSibling / nextElementSibling：(下一个节点);
previousSibling / previousElementSibling：（上一个节点）；
parentNode：（元素的父节点）
offsetParent：（元素有定位属性的父节点）
offsetLeft / offsetTop

js中的高度属性 和页面可视高度 页面滚动高度

关于宽高的一些属性：
  clientHeight / clientWidth     （width + padding）
  offsetHeight / offsetWidth     （width + padding + border）
  crollHeight / scrollWidth       (实际内容高度/宽度，有滚动条时会比client大/没有滚动或者隐藏时与client一样打)

  document.documentElement.clientWidth   // height；
滚动距离宽高：
  document.body.scrollTop    只适用于chrome浏览器
  document.documentElement.scrollTop 适用于其他浏览器

  document.onclick = function(){
    var scrollTop = document.documentElement.scrollTop || document.body.scrollTop;

  };

字符串的操作：

长度： str.length;下标： str[1];可读不可写；

返回某位置的字符：str.charAt(  ); //括号里写要找的位置

返回字符编码：str.charCodeAt( ); //括号里写要找的位置

返回字符：String.formCharCode( , ); //括号里面写编码 可多个

返回某个字符位置（左→右）：str.indexOf( ); //括号里面写某个字符 找不到返回-1|
（‘aa’,5） //从第5位开始找aa，第二参数为负数，当成0

返回某个字符位置（右→左）：str.lastIndexOf( );

截取字符串：str.substring( ); // (0,5)从0到5;(5,0)从0到5;(-2,5)从0到5
         ：str.slice( );     // (0,5)从0到5;(5,0)找不到；(-2,5）倒数第二位

拆分字符串：str.split( ); // ('.')以.为界限分割，返回一个数组；第二参数为长度

转换成大写：str.toUpperCase();

转换成小写：str.toLowerCase();

数组的操作：

定义：var arr = [1,2,3];  //  var arr = new Array(1,2,3);

数组长度 ： arr.length    // 可读可写

添加值： arr.push();     // 后面添加，返回数组长度
        arr.unshift();  // 前面添加，返回数组长度

删除： arr.pop();    // 删除数组最后一个，返回删除的那个
      arr.shift();  // 删除数组第一个，返回删除的那个
splice ：arr.splice(  );  //(0,1); 从第0位开始删除1个,返回删除的那个
                            //(0,1,'a'); 把第0位开始的第一位替换为'a',
                           // (0,0,'a'); 在第0位后面添加'a'
排序：arr.sort();  //  按照字符编码来排序
                       //  可传函数参数 
                  //  倒序 可以用reverse

两数组拼接：concat();  //  arr1.concat(arr2)

变为字符串：join();  //arr.join( 'a' )以a为界限符拼成字符串 即  "arr[1]aarr[2]"

# 基础常识 #

  1. js中命名通常o开头（意为object）

  2. js里不能直接控制标签属性class 即不能直接 .class 要用 .ClassName

  3. js里不能直接样式属性 float 即不能直接 .style.float 

    要用(兼容)  ie： .style.styleFloat   
            非ie： .style.cssFloat

    不过通常情况下涉及到float一般用设置className在内部样式中来解决

  4. js里获取到的路径和颜色不能用作判断！因为和实际写代码的不同 路径获取的为绝对路 径颜色获取的为rgb(不同浏览器情况不同)

  5. js里当函数自己执行的时候（即没有设置对象事件的时候） this指的对象是 window

    （window是js里最基本的对象 通常情况下都是省略的 除非！:window.resize时 不能省略）
