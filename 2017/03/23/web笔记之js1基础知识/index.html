发表于 2017-03-23 | 分类于 web笔记js 
# 基础常识 #

  1. js中命名通常o开头（意为object）

  2. js里不能直接控制标签属性class 即不能直接 .class 要用 .ClassName

  3. js里不能直接样式属性 float 即不能直接 .style.float 

    要用(兼容)  ie： .style.styleFloat   
            非ie： .style.cssFloat

    不过通常情况下涉及到float一般用设置className在内部样式中来解决

  4. js里获取到的路径和颜色不能用作判断！因为和实际写代码的不同 路径获取的为绝对路 径颜色获取的为rgb(不同浏览器情况不同)

  5. js里当函数自己执行的时候（即没有设置对象事件的时候） this指的对象是 window

    （window是js里最基本的对象 通常情况下都是省略的 除非！:window.resize时 不能省略）

数据类型

typeof 10;  // number 数字
typeof 'a我'  // string 字符串
typeof true // boolean 布尔值 true / false
typeof function(){} // function 函数
typeof [1,'2','3'] // object 对象
typeof document // object 对象
typeof {} // object 对象
typeof null; // object 对象
typeof undefined // undefined 未定义

得到布尔值boolean 用比较符

<  > == 比较值  例如  10 == '10'     true
!= <= >=  比较值
===   !== 不仅比较值，数据类型也比较
！！！！两个字符串比较大小的时候，不会比较值的大小，比较的是 ANSII码   
例如：  3>:"20"   true  因为3的ansii大于2

js里运算都是 从 左 往 右

加号两边有字符串或者数组 最后的组合项都是字符串 

例：a=5 b=6  alert（“a+b=”+a+b） 结果：a+b=56 字符串string

减乘除余不同 字符串与数字组合运算结果都是数字 

例： a="5"  b=6  alert(a-b) 结果：-1 数字number

多用于隐式类型转换 即 将数字数组变成字符串 或者将字符串变成数字类型

显式类型转换

Number();  把一个数据转换成 number类型
parseInt();  从第一位开始强制取整数
parseFloat(); 从第一位开始强制取数字
NaN :  not a number  不是数字    // NaN数据类型是  number 相当于先把数据Number()一次
NaN和自己都不相等
isNaN(); 讨厌数字  //通常纯数字 或者纯 数字型的字符串 会返回false
如果是NaN，返回true
否则，返回false;
先把参数用Number转换一下，再判断

三目运算

if ( 2<3 )
{
    alert(2);
}
else
{
    console.log(3);
}
可以写成：2<3?alert(2):console.log(3);
或者 if ( 2<3 )alert(2);
     alert(3);
或者 alert( 2<3?2:3 );
