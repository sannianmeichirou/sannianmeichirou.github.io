web笔记之js4 DOM

发表于 2017-04-02 | 分类于 web笔记js

DOM节点相关操作:

NodeType:节点类型有12种（常见的有 元素节点1、文本节点3、属性节点2）；
childNodes：（子节点集合）
             IE678只包含元素，其他包含文本和元素（包括非标准元素）；
             可以用 .length得到长度；
children：（只包含元素节点，推荐使用）；
firstChild：（第一个子节点）IE678只会是元素，其他可能是文本或元素；
firstElementChild：IE678没有这个属性，其他第一个元素节点；
lastChild / lastElementChild : （最后一个子节点）；
nextSibling / nextElementSibling：(下一个节点);
previousSibling / previousElementSibling：（上一个节点）；
parentNode：（元素的父节点）
offsetParent：（元素有定位属性的父节点）
offsetLeft / offsetTop

js里删除节点和创造节点

删除元素节点 .removeChild()：  //删除子节点
例：oDiv.parentNode.removeChild( oDiv ); 注（js里只能删除子节点，删除节点前必须找到父节点）
创建元素节点 .appendChild();
例：var obj = document.createElement('div'); //createElement 创建节点
    oDiv.appendChild( obj )                // appendChild 将创建的节点放入oDiv中

js中的高度属性 和页面可视高度 页面滚动高度

关于宽高的一些属性：
  clientHeight / clientWidth     （width + padding）
  offsetHeight / offsetWidth     （width + padding + border）
  crollHeight / scrollWidth       (实际内容高度/宽度，有滚动条时会比client大/没有滚动或者隐藏时与client一样打)
页面可视区宽高：
  document.documentElement.clientWidth   // height；
滚动距离宽高：
  document.body.scrollTop    只适用于chrome浏览器
  document.documentElement.scrollTop 适用于其他浏览器    
滚动高度兼容：
  document.onclick = function(){
    var scrollTop = document.documentElement.scrollTop || document.body.scrollTop;
    alert( scrollTop );
  };

js中 表单

xx.value       表示表单的值
input的name   在获取节点时可以document.getElementsByName()
               同时在form里面，可以直接 .(name名) 去获取元素节点 例如 oForm 是form元素 aa是input的name 则可以直接oForm.aa
checked：          用于radio（单选） 和checkbox      
                选项选中则    xx.checked = true 不选中则为false
                xx.checked 可以设置值 
               xx.checked='1'或者   
               xx.checked=function(){};
                 都表示true 但属性为自定义属性 
                 要用removeAttribute('checked')去除和getAttribute('checked')获取值；
                 注！ input 的name可以相同  但是一般情况下只在type = "radio" 时用相同的name 其他type均不用    
.focus()：      获得焦点
.onfocus        获取焦点事件 当获得焦点时触发 
.blur();          失去焦点  
                  注！.onfocus .onblur 事件必须是在能有焦点的元素才能触发
.onblur        失去焦点事件  当失去焦点时触发     例如input a标签 一些按TAB键能切换锁定的可以 一般div不行（写了事件也不触发） 
.onchange      内容改变事件 当内容改变的时候触发：
               (内容)text  :  失去焦点时，如何和得到焦点是的value不一样，就触发
               (单选)radio :  点击自己改变自己选中状态时才触发
              （多选）checkbox : 点击自己改变自己选中状态时就触发
              （选择列表））select : 改变option时触发
