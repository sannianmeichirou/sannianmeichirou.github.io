js中的正则表达式

一个匹配规则 （用来检测字符串满不满足这个规则的东西）

规则定义 ###：

    var reg = /qwe/;                 /匹配规则/                  
    var reg = new RegExp('qwe');     new RegExp（'匹配规则'）  
reg.test()   检测是否满足规则，如果满足规则，返回true，否则返回false
str.match()  满足匹配规则，把匹配成功的内容放到一个数组里面返回数组，否则返回null    
例：
       var reg = /qwe/;       匹配‘qwe’字符串
       var str = 'qweaaaaa';
       alert ( reg.test(str) );    //true        
       alert ( str.match(reg) );  // [qwe]

转义 \

    \s 单个空格            \S  非空格          均指的单个
    \d 单个数字            \D  非数字        
    \w  字符（字母 数字 _）     \W  非字符
    \b  独立部分（空格 开始 结束 -）\B  非独立部分 
    \跟上特殊意义的字符  正则里面的特殊字符 \ / { } + . ? * ( ) [ ] ^ $   \加上特殊符号  相当于匹配这些特殊符号
    \r 换行        \n 回车      \t 制表符相当于按一次tab
例：
       var reg = /\d/;  匹配单个数字        var reg = /\//;     匹配特殊符号/
       var str = 'qb2ca1';                var str = 'qb2/ca1';
       alert( str.match( reg ) );  //[2]        alert( str.match( reg ) );  //[/]

量词：{}

        {1,5}     :  1 <= n <= 5     1个到5个之间
    +    表示{1,} :  n >= 1    一个及以上    
    *    表示{0,} :  n >= 0    零个及以上
    ?    表示{0,1}:  0 <= n <= 1 零个或者一个
例：
       var reg = /w+/;  匹配一个以上的w
       var str = 'qqqwwwadqqw';    
       alert( str.match( reg ) );  //[www]


标识：    /条件/标识  标识写在结尾/后面 
    g   :   全局匹配 （从头到尾匹配完）  没有g则一次匹配成功后结束 有g则全局匹配成功后结束
    i   :   不区分大小写
例：    
    var reg = /w+/gi;    全局匹配一个及以上的w 且不区分大小写 
    var str = 'qqqwwWadqqw';
    alert( str.match( reg ) );  //[wwW,w]

子集： ()

    ()内为子集内容 匹配时子集作为整体匹配    注意！当有子项，并且没有全局标识的时候，match返回的数组的第二项代表子集内容
例：
    var reg = /(qw)+(ss)/;        //匹配子项1 (qw) 匹配子项2 (ss)        
    var str = 'qwqwssqqwsqwqwqwss';
    alert( str.match( reg ) );     //[qwqw,ss,qw,ss] 后面的qw,ss 为返还的子集内容

字符集： [ ]

    [0-9]  ： 从0到9    [1-5]  :  从1到5        
    [a-z]  :  a - z        [f-k]  :  f - k
    [A-Z]  :  A - Z
    []里内容为其他字符时候 表示或者
    字符集里面的 除了转义\  其他的 特殊字符 不特殊  是单纯的没有意义的字符            
例：
    var reg = /[(+q/{\de1,5}w)]/g;   全局匹配 （或者+或者q或者/或者{ 或者 单个数字 或者e或者1或者，或者5....
    var str = '(q+d{wdeq/w3fds\\';
    alert( str.match( reg ) );     // [(,q,+,{,w,e,q,/,w,3]

其他小知识点综合:

.   任意字符    .+表示所有均可匹配
^   开始   在字符集[]里面使用是  非
$   结束
var reg = /^qww/;    匹配以q为字符串开头并且后面跟着ww 
var str = 'qwwerqwweqqw';    
alert( str.match( reg ) ); // [qww]
或者： []单个字符的或者
    |  或者  写在外面  多个字符或者
例：var reg = /qqq|www/; 表示qqq或者www
    var reg = /qqq(aa)|www/; 表示 qqq(aa) 一个整体 或者www
    var reg = /aa(qqq|www)/;  表示aa  和 子集qqq 或者 子集www        
replace  str.replace( reg , strA );     把匹配成功的东西替换成strA;
strA : 可以是一个字符串，也可以是一个匿名并且带return的函数
       当是函数的时候：
              可以接收形参
              return 什么 替换成什么
    function( 形参1 , 形参2 , 形参3  , …… ){
    return xxxx;};
    形参1 表示匹配成功完整规则项（若有或者 则是哪项成功表示哪项 如果全局匹配则是匹配多次 表示多次运行function每次形参1都不同）
        形参2 表示规则第1个子集项                                  形参1总是表示匹配成功的规则整体
    形参3 表示规则第2个子集项....
例：
    var reg = /qq(w)|qq/g;
    var str = 'werqqwweqqr';
    var b = str.replace( reg , 'bb' );        // b: 'werbbwebbr'
    var a = str.replace( reg , function(q1 ,q2){
        console.log('q1'+':'+q1);    出现两次console 第一次 q1:qqw q2:w    
        console.log('q2'+':'+q2);            第二次 q1:qq  q2:undefined 
        return '123';                
    });                    // a:'wer123we123r'

QQ号的规则：

   只能是数字
   长度 ： 5  10
   不能以0开头            
var regQQ = /^[1-9]\d{4,9}$/;

邮箱的规则： xxx@xx.xxx.xx

var regMail = /^\w+@[0-9a-zA-Z]{2,}(\.[a-zA-Z]{2,}){1,2}$/;
   以多位字符开头 @ 2位以上的数字或者字母或者组合 一个或者两个子集（ . 加上 2位以上的字母的共同组合 ）

手机号码：

var regTel = /^1[34578]\d{9}$/;

身份证：

      var regID = /^[1-9]\d{16}[0-9xX]$/;
