ajax

在不刷新页面的情况下 对页面内容进行局部更新

ajax 基本案例：

    oBtn.onclick = function(){
            var ligoudaner = new XMLHttpRequest();
            ligoudaner.open( 'get' , 'firstAjax.txt' , true );
            ligoudaner.send();
            ligoudaner.onreadystatechange = function(){
                if ( ligoudaner.readyState == 4 )
                {
                    alert( ligoudaner.responseText );
                }
            };

            第一步：创建ajax对象
                var xhr = new XMLHttpRequest();
                //var xhr = new ActiveXObject('Microsoft.XMLHTTP');

            第二步：请求准备
                xhr.open( 'get' , 'firstAjax.txt' , true );
                    参数1：请求方式 get/post
                        在form提交的时候：
                            get会把传输数据放在url后面
                            post只会把数据放在头文件里面
                    参数2：请求路径
                    参数3：是否异步
                        true 异步 请求的过程不会影响后面程序的执行
                        false 同步

            第三步：正式发送请求
                xhr.send();

            第四步：监听请求状态
                xhr.readyState ：请求状态码
                    0 ：请求还没发生（open执行之前）
                    1 ：请求已经建立，还没发送（执行了open）
                    2 ：请求已经发送，正在处理（执行了send）
                    3 ：请求处理中，有一部分数据可以用，但还有没有完成的数据。
                    4 ：请求完全完成

onreadystatechange ：请求状态码发生改变的时候触发

xhr.responseText : 返回的数据

xhr.status : HTTP状态码  （网上可查阅 2开头表示成功....）

eval( '(' +a+ ')' )   a是string  eavl：将数据以js代码形式运行（将字符串变成可运行的代码类型）
例：
    var a = "[{name:'abc',age:18} , {name:'ddd',age:38}]";
    var b = eval( '(' +a+ ')' );
