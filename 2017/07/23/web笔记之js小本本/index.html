js 中 [] 除了数字   中控号中只能是字符串 或者是一个 代表字符串的变量
例： box.style.width 与 box.style["width"]
    或者 a = 'width';
    box.style[a]  三者是一样的    
    但是不能box.style.a  不能点a 因为a是变量不是style的属性

i=0 i++
if (i>=xx.length)
{ i=0;}
 这种形可以简写为   i %= xx.length   例如
 if（i>=5){i=0}; 简写：i %= 5;
                 分析 当 i= 0 时 i %= 5 //结果： i=0
                     当 i= 1  时 i %= 5 //      i=1
                .....当 i = 4 时 i %= 5 //      i=4
                     当 i= 5  时 i %=5 //  5%5 =0

js中触发事件 当前一个变化 前一个效果消失逻辑
以onclick事件为例：
          var index = 0;
          for(var i=0;i<xx.length;i++)
          {
            xx.i = i;  //自定义属性 将所有的i赋给xx.i集合、、
            xx[i].onclick = function(){
             xx[index].ss = ''; //首次第0个效果为空 后面index则是前一个[i] 效果为空//
             index = this.i;    //将当前的[i]赋值给index 下一次点击事件 则index就是前一个[i] 对应上一个步//
             this.ss = 'aa';    //当前效果为aa//
             }；
           }；

拓展  关于 = 的赋值和引用
        赋值： （ number string boolean …… ）        不改变原来的元素 仅仅赋值
        引用： （ object  function ）    引用相同的内存地址  现元素的改变 会改变原来元素
    例:        
        var a = 10;            var c = { num:10 };
        var b = a;            var d = c ;
        b++;                d.num++;
        alert(a);            alert(c.num);    
        // 10   这是赋值        // 11  这是引用
