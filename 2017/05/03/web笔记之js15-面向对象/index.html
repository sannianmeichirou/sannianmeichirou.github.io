面向对象

面向过程： 流程式的写法

面向对象： 所有的程序执行都从对象出发

对象拥有属性和方法

        1.所有的功能都从对象出发，已达到一个对象实现某种功能的效果
        2.面向对象支持扩展（继承）    
在函数执行前面加上new：
                1.函数内部自然会产生一个对象，this指向这个对象
                2.函数默认返回产生的这个对象
    function Person(name){            function Person(name){
        this.name = name;            var obj = {};
    };                   与        obj.name = name;
    var a = new Person('123');            return obj;
                            };
                        var a = Person(123);
                意思差不多  不过new内部产生的this指向obj

构造函数:

面向对象所写的函数function 构造函数名的头字母一般都是大写
        function Person( name ){
            this.name = name;
            this.showName = function(){
                alert( this.name );
            };
        };

        var obj1 = new Person( 'onmyway' );
        var obj2 = new Person( '左缺' );

        alert( obj1.showName == obj2.showName );
            结果为false

原型： prototype

只有构造函数有原型
原型的方法里面的this指向的是执行的时候的对象                
    在写面向对象的时候：
        私有属性写在构造函数里面
        公共属性写在prototype里面

function Person( name ){
    this.name = name;
};
Person.prototype.showName = function(){
    alert( this.name );
};

var obj1 = new Person( 'onmyway' );
var obj2 = new Person( '左缺' );

alert( obj2.showName == obj1.showName )
        结果为true  公有属性在原型里所用的内存地址是相同的

包装对象：

// 数组： Array
    var arr = [1,2,3];

    Array.prototype.push = function(){
        var argu = arguments;
        for ( var i=0;i<argu.length;i++ )
        {
            this[this.length] = argu[i];
        }
    };

    arr.push(4,8,9);

    alert( arr );

// 对于非对象的数据类型，是由包装对象产生的
例：字符串
    var str = 'qwer';
    alert( str.charAt(2) );  //e
str为字符串 charAt(2)表示查找字符串第3个字符的方法        
按道理 str为字符串不是对象 是不能 .方法的  只有对象才能 .方法    
但确实有 这是为什么
因为包装对象
    js中存在 String.prototype.charAt()
    这说明在使用 str.charAt() 的时候产生了一个面向对象 
    而这个面向对象的构造函数便是String 
    String的原型有charAt方法 所以str可以试用.charAt
    但是    这个str产生的面向对象与正常的面向对象不同
    它产出的对象this 在产生之后就会删掉
例     
    var str = 'qwer';
    str.number = 10;
    alert( str.number );  //undefind
   按道理 str是字符串不是对象 不能.属性的
         但结果是undefind而不是语法错误 
   就是因为 str 的包装对象
   str.number 产生了一个面向对象 (其实是包装对象）
   但是这个对象马上就被删掉了 
   它的构造函数String里面没有number属性 原型里面也没有number方法
   所以 显示结果为undefined

原型链 ：proto

   对象和它的构造函数原型的关系
例1    function A(argu){
        this.number = argu;
    };

    A.prototype.number = 10;

    var objA = new A( 5 );

    alert( objA.number );  // 5
        如果 对象本身有属性 那么以本身属性为准
    如果对象没有 属性那么以原型属性为准    

例2
    function A(argu){
        //this.number = argu;
    };

    Object.prototype.number = 20;

    //A.prototype.number = 10;

    var objA = new A( 5 );

    alert( objA.number ); // 20
  A.prototype 本身自己就是一个对象  因为可以.属性和.方法
  那么A.prototype 肯定就有一个构造函数（假设是B） 和 原型 因为对象都是由面向对象产生的
  那么B.prototype 也是一个对象 也有一个构造函数 和 原型
          .......  有很多
  最终的 对象 也就是最后最根本的那个对象  在js中就是Object
  所以 如果A本身没有 number属性 A的原型A.prototype也没有number属性
          就会一级一级往上找  最终找到Object 
  所以在这里定义了 Object.prototype.number = 20
  所以objA.number 也是20

继承：

    儿子继承爸爸；
    儿子可以用爸爸的方法和属性；
    儿子的改变不影响爸爸；
function Person( name , age , marry , sex ){
            this.name = name;
            this.age = age;
            this.marry = marry;
            this.sex = sex;
        };

        Person.prototype.showName = function(){
            alert( this.name );
        };


        // 通过继承创造一个子函数
        function PersonChild(  pArr , weight ){
            Person.apply( this , pArr );
            this.weight = weight;
        };    
var oP = new PersonChild( ['abc' , '18' , false , 1] , 60 );

    注：    这里用Person.apply( this , pArr );这个方法 让PersonChild函数继承了Person 函数的属性

继承原型函数 两种方法

1 递归 ：

类似阶乘 ！ 例如： 4！= 4*3！ 
                = 4*3*2！
                = 4*3*2*1
var a = factorial( 4 );
    alert( a );

    function factorial( num ){
        if ( num == 1 )
        {
            return 1;
        }
        return num * factorial(num-1);
    }            
   1:f（4）.return 4*f(3)   2:f(3).return 3*f(2)    3:f(2).return 2*f(1)    4:f(1).return 1
   先递：  4*f(3)  3*f(2) 2*f(1) f(1)
   后归：  先第4步return 1 给 第3步 的f(1) 计算出2*f(1)    然后将结果return给f(2) 计算出第二步的结果 再ruturn给第一步计算 
       最后得到4*f(3)

用递归得到的克隆构造函数：

    function Clone( obj ){
        for ( var key in obj )
        {
            this[key] = (typeof obj[key]=='object')?new Clone( obj[key] ):obj[key];
        };
    如何使用 得到继承原型:
        PersonChild.prototype = new Clone(Person.prototype);

2 利用面向对象时 创造对象的特点 进行继承

例： function P(){};
     P.prototype.aaa = function(){};    
     var a = new P();
a对象 可以得到 P.prototype的所有属性 即 a.aaa 成立
    同时 a的改变 不影响 P.prototype  即a.b = x ;与P.prototype 无关

function inherit( obj ){
    function Fn(){};
    Fn.prototype = obj;
    return new Fn();
}        
PersonChild.prototype = inherit(Person.prototype);
